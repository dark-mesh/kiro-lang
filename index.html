Module System

Any .kiro file in the same directory can be imported. Access exported members with qualified syntax. Kiro also ships with an embedded standard library (std_fs, std_net, std_env, std_io, std_time) baked directly into the binary for zero-configuration portability.

// math.kiro
fn add(a: num, b: num) {
    return a + b
}

// main.kiro
import math

fn main() {
    var result = math.add(10, 20)
    print result
}
main()

Host Modules (Rust FFI)

Zero-friction access to the Rust ecosystem. Declare external functions with the rust keyword. The logic lives in a centralized header.rs glue file using the kiro_runtime crate.

// Kiro side
rust fn read_file(path: str) -> str!

// header.rs (Rust glue)
use kiro_runtime::{RuntimeVal, KiroError};

pub fn read_file(args: Vec<RuntimeVal>)
    -> Result<RuntimeVal, KiroError> {
    let path = args[0].as_str()?;
    match std::fs::read_to_string(path) {
        Ok(c) => Ok(RuntimeVal::from(c)),
        Err(_) => Err(KiroError::new("NotFound")),
    }
}
